# MORAT — Digital Lookbook

Proyecto web desarrollado como actividad académica para la asignatura  
**Análisis y Desarrollo de Software**

---

## 📌 Descripción

Este proyecto es una página web tipo *lookbook digital* inspirada en la banda MORAT.  
La página presenta una estructura visual moderna, organizada y limpia, combinando diseño, contenido informativo y composición gráfica.

Se desarrolla como ejercicio práctico para aplicar fundamentos de desarrollo web y diseño de interfaces.

---

## 🎯 Objetivo del proyecto

- Aplicar estructura semántica en HTML  
- Implementar estilos con CSS  
- Usar Grid Layout y Flexbox  
- Organizar un proyecto web correctamente  
- Separar estructura y diseño  
- Aplicar diseño visual básico  
- Usar buenas prácticas de organización de archivos  

---

## 🧱 Tecnologías utilizadas

- HTML5  
- CSS3  
- Google Fonts  
- Grid Layout  
- Flexbox  
- Diseño responsive básico
